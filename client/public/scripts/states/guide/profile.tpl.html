Search resume profiles

  1. Click the search resumes button on the side nav.
  2. Use the pager by scrolling to the page bottom to see more listings.
  3. Open the listings filter by clicking the filter icon in the subheader.
  4. After the listings filter is opened use the job types filter or the job position filter to narrow your search.
  5. It is possible to sort resume profiles by proximity.
  6. There are several popular ports to choose from in the select location select box.
  7. If your location is not contained in the select location select box, use the Google places autocomplete search to look up your location. Once you select your location, it will update the proximity sort.
  8. Once the filters and proximity sort is chosen, click the submit filter button to activate.

Create your resume profile

  1. To create your resume profile either open the user menu and click resume profile or go to the account dashboard and click create under my resume profile section.
  2. Fill out every field in the form. Click the submit button when all require fields are completed.
  3. Once created you can check in once every 24 hours. This will push your active resume profile to top of the list so it doesn't become stale. On the account dashboard page click the check in button to move your resume profile to the top of the list.
  4. On the account dashboard page you can activate and deactivate your profile at any time.
